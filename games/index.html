Game Boy Color RPG

A work-in-progress project, programmed from scratch in Z80 assembly. While it's not fully playable yet, it contains a basic sound engine, variable-width text display, a fixed decompressor, a basic overworld game loop, and more.

Dungeon of Doom

Created as part of my first-year Java programming coursework. From the primitive command-line game that were initially given, I added multiplayer (via networking), a graphical interface with smooth animations, and a computer player.

Arduino Judge

Created as part of my first-year systems architecture coursework. Working together with a friend, we aimed to recreate the excitement of the original Game & Watch Judge, but through microgames that revolve around LEDs, piezo speakers, and LDRs.
